# 🌿 Enciclopedia de las Hierbas Mágicas

¡Bienvenido al mundo de la herbolaria mística! Esta es una revista digital interactiva basada en el conocimiento de **Scott Cunningham**, diseñada con un efecto de "pase de página" para una experiencia inmersiva.

## ✨ Características
- **Efecto de Libro Real:** Pasa las páginas haciendo clic o arrastrando las esquinas.
- **Diseño Místico:** Estética premium con fuentes clásicas y fondos atmosféricos.
- **Contenido Detallado:** Guía sobre géneros, planetas, elementos y usos mágicos de decenas de hierbas.
- **Formato Compacto:** Ideal para leer como un libro de bolsillo digital.

## 📖 Cómo Leer
Si estás viendo esto en GitHub Pages, simplemente desplázate y usa las esquinas de las páginas para navegar.

## 🛠️ Hecho con
- **HTML5 / CSS3** (Vanilla)
- **JavaScript**
- **PageFlip.js** para el efecto de libro interactivo.
